# Template Selection

Modern React SPA starter with backend

Use when:
- Universal apps with backend persistence
- Apps with dashboards for external data
- Simple, fast Vite projects
- Full-scailed Backend, databases etc
- Heavy backend/server needs
- User Authentication required

Avoid when:
- Client-side only apps with no backend persistence
- Simple static apps

Built with:
- React Router, ShadCN UI, Tailwind, Lucide Icons, ESLint, Vite, Andromo Convex, Convex Auth
